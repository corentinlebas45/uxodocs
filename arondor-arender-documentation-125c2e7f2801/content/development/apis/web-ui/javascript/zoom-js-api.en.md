### Change zoom

- Object: getARenderJS().getZoomJSAPI()

    | Function            | Description                                      |
    | ------------------- | ------------------------------------------------ |
    | askZoomIn()         | Zoom                                             |
    | askZoomOut()        | Zoom out                                         |
    | askZoomFullWidth()  | Zoom full width                                  |
    | askZoomFullHeight() | Zoom full height                                 |
    | askZoomFullPage()   | Zoom full page: adapted to both width and height |
