### Switch to Full screen

- Object: getARenderJS().getFullScreenJSAPI()

    | Function             | Description                 |
    | -------------------- | --------------------------- |
    | askOpenFullScreen()  | Activate full screen mode   |
    | askCloseFullScreen() | Deactivate full screen mode |
