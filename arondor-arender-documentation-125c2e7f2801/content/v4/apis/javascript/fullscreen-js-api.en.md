### Switch to Fullscreen

- Object: getARenderJS().getFullScreenJSAPI()

    | Function             | Description                |
    | -------------------- | -------------------------- |
    | askOpenFullScreen()  | Activate fullscreen mode   |
    | askCloseFullScreen() | Deactivate fullscreen mode |
