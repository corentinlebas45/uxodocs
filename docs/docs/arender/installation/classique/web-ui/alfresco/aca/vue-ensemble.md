# Alfresco ACA - Vue d'ensemble

Integration d'ARender dans Alfresco Content App (ACA).

## Présentation
ACA (Alfresco Content App) est l'interface moderne d'Alfresco basée sur Angular.

## Avantages ACA + ARender
- Interface utilisateur moderne
- Performance optimisée
- Annotations temps réel
- Expérience mobile

## Architecture
```
[ACA Frontend] → [ARender Extension] → [ARender Web-UI] → [Rendition Server]
```
