### Mettre en plein écran

- Objet : getARenderJS().getFullScreenJSAPI()

    | Fonction             | Description                   |
    | -------------------- | ----------------------------- |
    | askOpenFullScreen()  | Active le mode plein écran    |
    | askCloseFullScreen() | Désactive le mode plein écran |
